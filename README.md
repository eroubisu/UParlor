# Parlor — 终端游戏厅

在终端里和朋友下棋、打牌、聊天。

## 它是什么

Parlor 是一个运行在终端里的**多人在线游戏大厅**。连上服务器，登录账号，就能和其他玩家一起玩。

界面用键盘操作，支持 Vim 风格快捷键，不需要鼠标。

## 安装

```
pip install gamelobby
```

## 启动

```
gamelobby
```

首次运行会连接到公共服务器，注册账号后即可游玩。

## 操作方式

| 按键 | 功能 |
|------|------|
| `/` | 打开指令栏 |
| `Esc` | 返回 / 关闭 |
| `Tab` | 切换指令分类 |
| `↑` `↓` 或 `j` `k` | 上下选择 |
| `Enter` | 确认 |
| `Space` | 快捷菜单 |

## 功能

- **游戏大厅** — 浏览可用游戏，输入 `play` 进入
- **实时聊天** — 大厅内所有人可互相聊天
- **个人系统** — 个人资料、头衔收集、背包物品
- **段位排名** — 对战积累经验，提升段位

## 自建服务器

```
cd Server
pip install .
python server.py [端口号]
```

客户端启动时可指定服务器地址：

```
gamelobby --host 你的IP --port 端口号
```
