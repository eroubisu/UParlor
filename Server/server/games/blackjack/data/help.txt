## welcome 21点
◆ 21点 (Blackjack)

  手牌点数尽量接近 21 点但不超过，打败庄家即可获胜。
  2-6 人游戏。

  create 创建房间 → invite 邀请 → start 开始
  help 查看详细帮助

## rules 游戏规则

  目标: 手牌点数尽量接近 21 点但不超过。
  牌面 2-10 按面值算，J/Q/K 算 10，A 算 1 或 11。

  ◆ 特殊规则
    天然 21 点（前两张 = 21）直接获胜，赔率 1.5 倍。
    庄家 16 点以下必须补牌，17 点以上停牌。

## controls 操作方法

  轮到你时从以下操作中选择：

  ◆ 可用操作
    hit       要牌（再拿一张）
    stand     停牌（不再要牌）
    double    加倍（加倍下注，只能再拿一张牌）

  爆牌（超过 21 点）自动判负。

## commands 指令列表

  ◆ 房间管理
    create    创建房间
    rooms     查看房间列表
    join      加入房间
    invite    邀请好友
    kick      踢出玩家
    start     开始游戏

  ◆ 游戏中
    hit       要牌
    stand     停牌
    double    加倍

  ◆ 通用
    back      返回上一层
    help      查看帮助
