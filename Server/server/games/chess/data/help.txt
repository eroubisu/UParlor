## welcome 国际象棋
◆ 国际象棋

  标准国际象棋规则，两人对弈。白方先行。
  支持人机对战（easy/normal/hard）。

  create 创建房间 → bot 添加机器人 → start 开始
  help 查看详细帮助

## rules 游戏规则

  标准国际象棋规则，两人对弈。
  白方先行，交替走子，将杀对方国王即获胜。

  ◆ 胜负判定
    将杀（checkmate）→ 对方获胜
    和棋条件：双方同意、逼和（stalemate）、重复局面

## pieces 棋子走法

  ◆ 各棋子走法
    ♔ 王 (King)     任意方向一格
    ♕ 后 (Queen)    任意方向任意格
    ♖ 车 (Rook)     横/竖任意格
    ♗ 象 (Bishop)   斜线任意格
    ♘ 马 (Knight)   L 形跳跃（2+1），可越子
    ♙ 兵 (Pawn)     前进一格，首步可两格，斜前方吃子

  ◆ 特殊走法
    王车易位    王和车均未动过，中间无子，王不经过被攻击格
    吃过路兵    对方兵首步走两格时，可用己方兵斜吃
    兵升变      兵到达底线自动升变（默认皇后）

## controls 操作方法

  使用 UCI 坐标输入走子：
    起始格 + 目标格，例如 e2e4

  升变指定棋子（默认皇后）：
    例如 e7e8q（q=后, r=车, b=象, n=马）

  坐标说明：
    列 a-h（左到右），行 1-8（白方底部到顶部）

## commands 指令列表

  ◆ 房间管理
    create    创建房间
    rooms     查看房间列表
    join      加入房间
    invite    邀请好友
    bot       添加机器人（easy/normal/hard）
    kick      踢出玩家
    start     开始游戏

  ◆ 游戏中
    直接输入走子（如 e2e4）
    resign    认输
    draw      提议和棋
    accept    接受和棋
    reject    拒绝和棋

  ◆ 通用
    back      返回上一层
    help      查看帮助
