## welcome 斗地主
◆ 斗地主 (Dou Di Zhu)

  三人游戏，一副 54 张牌（含大小王）。
  叫分抢地主，先出完牌者获胜。

  create  创建房间
  invite  邀请好友
  start   开始游戏
  help    查看详细帮助

## rules 规则

  ◆ 游戏流程
    1. 发牌: 每人 17 张，留 3 张底牌
    2. 叫分: 轮流叫 0~3 分，最高者为地主
    3. 地主获得 3 张底牌（共 20 张）
    4. 地主先出牌，逆时针轮转
    5. 先出完所有牌者获胜

  ◆ 倍率
    基础倍率 = 叫分值（1/2/3）
    炸弹/火箭: 倍率 ×2
    春天: 地主赢且农民一张未出 → 倍率 ×2
    反春: 农民赢且地主只出一手 → 倍率 ×2

  ◆ 段位
    初心 → 学士 → 精英 → 大师 → 宗师 → 传奇
    每段三阶，累计段位点升降。
    胜: +8pt  负: -5pt（初心段不扣）
    段位点受倍率加成（×2 时胜 +16pt）

  ◆ 奖励（受倍率加成）
    胜: +150 经验  +40 金币
    负: +100 经验  -15 金币

  ◆ 机器人对局
    金币按单人比例发放
    初心~学士段: 段位点减半
    精英段及以上: 不涨段位点
    不扣段位点

## patterns 牌型

  ◆ 基本牌型
    单张          任意一张牌
    对子          两张相同点数
    三条          三张相同点数
    三带一        三条 + 一张单牌
    三带二        三条 + 一对
    顺子          ≥5 张连续单牌（3~A，不含 2 和王）
    连对          ≥3 对连续对子（3~A）
    飞机          ≥2 组连续三条
    飞机带翼      飞机 + 等量单牌或等量对子
    四带二        四张 + 两张单牌或两对
    炸弹          四张相同点数
    火箭          大王 + 小王

  ◆ 大小规则
    火箭 > 炸弹 > 普通牌型
    炸弹之间比点数
    普通牌型只能同类型同长度比较
    点数: 3 < 4 < ... < K < A < 2 < 小王 < 大王
    显示: B☆ = 大王，S☆ = 小王

## controls 操作与指令

  ◆ 叫分阶段
    bid       从子菜单选择叫分（0/1/2/3）

  ◆ 出牌阶段
    i → 序号  进入输入模式，输入手牌序号出牌
    i → p     不出（跳过）
    牌序号从 0 开始，对应手牌显示的编号
    系统自动识别牌型，非法出牌会提示
    需要接牌时，系统会列出可打的牌型供选择

  ◆ 房间管理
    create    创建房间
    rooms     查看房间列表
    join      加入房间
    invite    邀请好友
    kick      踢出玩家
    start     开始游戏

  ◆ 通用
    back      返回上一层
    help      查看帮助
