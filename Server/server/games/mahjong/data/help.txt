  四人对局，标准日本麻将规则。

  create 创建 → bot 添加机器人 → start 开始
  help 查看详细帮助

## rules 游戏规则

  日本麻将（立直麻将），136 张牌。
  目标：凑成和牌牌型（4 面子 + 1 雀头）。

  ◆ 基本概念
    面子: 顺子（三张连续同色）或刻子（三张相同）
    雀头: 一对相同的牌
    门前清: 未吃碰杠，全靠自己完成

  ◆ 荣和与自摸
    荣和（ron）: 别人打出你需要的牌
    自摸（tsumo）: 自己摸到需要的牌
    门前清限定和了方式更多

## actions 操作说明

  轮到你时输入手牌序号打出对应的牌（从 0 开始）。

  ◆ 他人打牌后可选动作
    ron       荣和（别人打出你能和的牌时）
    pon       碰（别人打出你有对子的牌时）
    chi       吃（上家打出你能顺子的牌时）
    pass      过（放弃吃碰和）

  吃牌时从子菜单选择组合。

  ◆ 自己的回合
    tsumo     自摸（摸到能和的牌时）
    riichi    立直（门前清听牌时宣言）
    输入序号  打出手牌

## yaku 常见役种

  ◆ 一番
    立直        门前清听牌宣言
    门前清自摸  门前清状态下自摸
    平和        门前清 + 全顺子 + 两面听
    断幺九      不含 1/9/字牌

  ◆ 二番
    一气通贯    同色 1-9 三组顺子
    三色同顺    三色相同数字顺子

  ◆ 三番及以上
    混一色      一种花色 + 字牌
    清一色      纯一种花色（六番）
    国士无双    13 种幺九牌各一 + 任一重复（役满）
