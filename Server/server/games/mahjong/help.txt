麻将

  日本麻将（立直麻将）

  使用 create 创建房间，invite 邀请好友，bot 添加机器人

  4人凑齐后 start 开始游戏
  轮到你时输入序号打出对应的牌

  动作指令（直接输入，不需要 /）:
    ron      荣和（别人打出你能和的牌时）
    pon      碰（别人打出你有对子的牌时）
    chi      吃（上家打出你能顺子的牌时）
               用法: chi 一万 二万（指定手中两张牌）
    pass     过（放弃吃碰和）
    tsumo    自摸（摸到能和的牌时）
    riichi   立直

指令

  create    创建新房间
  rooms     查看房间列表
  join      加入指定房间
  invite    邀请好友 (invite @玩家名)
  bot       添加机器人
  kick      踢出玩家
  start     开始游戏（房间内）
  riichi    宣言立直
  back      返回上一层
  help      查看帮助
