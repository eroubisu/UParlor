## explore 探索与移动

  ◆ 移动
    h / j / k / l    左/下/上/右移动（同 Vim 方向键）
    按住方向键可连续移动

  ◆ 进入与离开
    走到建筑门口时自动提示
    enter     进入建筑/区域
    back      离开当前建筑，返回小镇

  ◆ 传送阵
    地图上的紫色 %% 标记为传送阵
    首次到达时自动点亮
    interact  使用传送阵，前往已点亮的目的地（需花费金币）

  ◆ 其他
    recall    传送回地图出生点（需读条 4 秒）
    user      查看附近玩家
    talk      与 NPC 对话

## buildings 小镇建筑

  ◆ 商店
    铁匠铺      武器装备（buy/sell/forge）
    草药店      药草和药剂（buy/sell/brew）
    渔具店      钓鱼用品（buy/sell）

  ◆ 休闲
    酒馆        休息恢复 HP/MP（rest），打听消息（rumor）
    冒险者公会  任务系统（quest/board）

  ◆ 游戏
    棋牌室      麻将、斗地主
    赌场        21点、德州扑克
    棋馆        国际象棋
    文字馆      Wordle

  进入游戏建筑后用 play 选择游戏。

## shop 商店系统

  在铁匠铺、草药店、渔具店中可用：

  ◆ 购买与出售
    buy       查看可购商品列表
    sell      查看可出售的背包物品
    选择商品后自动确认购买/出售

  ◆ 锻造（铁匠铺）
    forge     查看可锻造配方
    需要材料和金币，成品直接进入背包

  ◆ 炼药（草药店）
    brew      查看可炼制药品
    治疗草 ×2 → 经验药水

## fishing 钓鱼

  ◆ 准备
    在渔具店购买鱼竿（fishing_rod）

  ◆ 钓鱼流程
    走到水边（蓝色瓦片旁）
    fish      抛竿
    等待鱼上钩（提示出现时）
    pull      收杆

  钓到的鱼可在商店出售。
