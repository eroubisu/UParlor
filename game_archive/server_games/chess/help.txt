国际象棋帮助

【房间系统】
  /create              创建房间（交互式选择段位场和时间控制）
  /rooms               查看可加入的房间
  /join <房间ID>       加入房间
  /invite @玩家名      邀请在线玩家
  /accept              接受邀请
  /bot                 添加机器人（仅友人场）
  /start               开始游戏（房主）

  段位场: 友人场 / 铜之间 / 银之间 / 金之间 / 玉之间 / 王座之间
  时间控制: blitz(3分+2秒) / rapid(10分+5秒) / classical(30分+10秒)

【走棋】
  /m e4                走一步（标准代数记谱法 SAN）
  e4                   也可直接输入（不加 /m）
  /moves               查看所有合法走法

  走法格式（SAN）:
    e4     兵到 e4
    Nf3    马到 f3
    Bxe5   象吃 e5
    O-O    王侧易位
    O-O-O  后侧易位
    e8=Q   兵升变为后

  也支持 UCI 格式: e2e4, g1f3, e7e8q

【查看】
  /board  /b           显示当前棋盘
  /history             走棋记录

【段位与战绩】
  /rank                查看段位详情
  /stats               查看战绩统计

【对局中操作】
  /draw                提出和棋
  /accept              接受和棋
  /decline             拒绝和棋
  /resign              认输
  /back                返回（对局中需确认）
  /quit                离开国际象棋
